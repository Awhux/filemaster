## ğŸš€ Getting Started

### Prerequisites

```bash
# Linux/Unix
sudo apt-get install build-essential  # Ubuntu/Debian
# or
sudo yum install gcc-c++              # RedHat/CentOS

# macOS
xcode-select --install
```

### Building

```bash
git clone https://github.com/Awhux/filemaster.git
cd filemaster
make clean && make
```

### Running

```bash
./filemaster [directory_path]
```
